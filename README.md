<h2 align="center">Installation</h2>

1. Clone the project:
```
git clone https://github.com/samuel3105/2Dgame_webSocket.git
```

2. Install dependencies:
```
cd 2Dgame_webSocket
npm install
```

3. Start server:
```
cd ./server
nodemon server.js
```
or 
```
cd ./server
node server.js
```

4. Start game:

Go to `http://localhost:3000/`
